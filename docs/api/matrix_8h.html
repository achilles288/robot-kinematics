Detailed Description

The header declaring 3x3 and 4x4 matrix structures and functions.

3x3 and 4x4 matrices. However, they have an exceptional having one row removed. The third block is always 0s and the fourth part a 1. The row is not used in computation for better performance. These matrix structures and functions are only for rotation and translation applications.

◆ rkMat4GetRotation()

rkEuler rkMat4GetRotation ( rkMat4  M)

Calculate the euler angles from the members of the matrix.

Calculation is done by applying functions to certain cells of the matrix. According to the formula derivation, there are 3 branches for the code. In case of Gimbal Lock, since cos(theta) is 0 and the default division by cos(theta) functions don't work, there use alternative approach.

Parameters
MTransform matrix
Returns
Euler angles
